static function bool():Bool

Return a random boolean value (true or false)

static function date(earliest:Date, latest:Date):Date

Return a random date & time from within a range. The behaviour is unspecified if either earliest or latest is null. Earliest and Latest are inclusive

static function enumConstructor<T>(e:Enum<T>):Null<T>

Return a random constructor from an Enum. Will return null if the enum has no constructors. Only works with enum constructors that take no parameters.

static function float(from:Float, to:Float):Float

Return a random float between 'from' and 'to', inclusive.

static function fromArray<T>(arr:Array<T>):Null<T>

Return a random item from an array. Will return null if the array is null or empty.

static function fromIterable<T>(it:Iterable<T>):Null<T>

Return a random item from an iterable. Will return null if the iterable is null or empty.

static function int(from:Int, to:Int):Int

Return a random integer between 'from' and 'to', inclusive.

static function shuffle<T>(arr:Array<T>):Array<T>

Shuffle an Array. This operation affects the array in place, and returns that array. The shuffle algorithm used is a variation of the Fisher Yates Shuffle

static function string(length:Int, ?charactersToUse:String = 'abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789'):String

Return a random string of a certain length. You can optionally specify which characters to use, otherwise the default is (a-zA-Z0-9)
