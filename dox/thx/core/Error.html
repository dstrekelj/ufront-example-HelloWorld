class thx.core.Error

Available on all platforms

Sub classes
AbstractMethod, NotImplemented, NullArgument
AbstractMethod

AbstractMethod extends Error and it is intended to be used inside those methods that are considered abstract. Abstract methods are methods that needs to be implemented in a sub-class.

NotImplemented

NotImplemented extends Error and it is intended to be used inside method that are drafted but still do not provide an implementation.

NullArgument

NullArgument is used to detect if arguments of methods are null or empty values.

Defines a generic Error type. When the target platform is JS, Error extends the native js.Error type.
