RemotingApi is a base class for setting up your Haxe remoting API context.

UFAsyncApi

A class that builds an API proxy of an existing UFApi. On the server it just wraps results in Futures. On the client it uses a HttpAsyncConnectionWithTraces to perform remoting. Constructor dependency injection is used to get the original API on the server or the remoting connection on the client. Usage: class AsyncLoginApi extends UFAsyncApi<LoginApi> {}
