package ufront.spadm

DBAdmin

An extension of DBAdmin that co-operates better with ufront.db classes Changes - Ignore classes with @noTable metadata - Find classes using CompileTime.getAllClasses() instead - Check relationships and add ManyToMany tables It has a handler() static method that is used in the exact same way as spadm.Admin.handler for processing the actual requests.
