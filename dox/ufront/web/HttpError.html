static function fakePosition(obj:Dynamic, method:String, ?args:Array<Dynamic> = null):{methodName:String, lineNumber:Int, fileName:String, customParams:Array<Dynamic>, className:String}

Generate a fake HaxePos position for a given class, method and args.

Useful for debugging async code when you're not sure where the error came from.

obj
  • The object (controller / module / handler) from which we derive the class name
method
  • the method name to use in our position
args
  • any arguments parsed to that method. If not given, an empty array will be used.

static function wrap(e:Dynamic, ?msg:String = 'Internal Server Error', ?pos:Pos = null):Error

Wrap an existing error into a Error

  • If it was an Error already, return it as is
  • If it was a normal exception, use code 500, with "Internal Server Error" as the message, the exception as the data, and the pos that call site for wrap() as the pos.
