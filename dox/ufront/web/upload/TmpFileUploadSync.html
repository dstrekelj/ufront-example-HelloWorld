class ufront.web.upload.TmpFileUploadSync implements FileUpload

Available on all platforms

A FileUpload implementation that works on SYS platforms (Neko, PHP etc).

All methods are performed synchronously, but work with the async interface.

Instance Fields

var originalFileName:String

The original name of this file on the client.

var postName:String

The name of the POST argument (that is, the name of the file input) that this file was uploaded with.

var size:Int

The size of the upload in Bytes

function new(tmpFileName:String, postName:String, originalFileName:String, size:Int):Void

Create a new TempFileUploadSync

Please note that "originalFileName" will be sanitised using haxe.io.Path.withoutDirectory().

function deleteTemporaryFile():Outcome<Noise, Error>

Delete the temporary file.

After doing this, other functions that rely on the temporary file will no longer work.

function getBytes():Surprise<Bytes, Error>

Get the current upload as a haxe.io.Bytes

Uses sys.io.File.getBytes(tmpFileName) behind the scenes.

function getString():Surprise<String, Error>

Get the current upload as a String

Uses sys.io.File.getContent(tmpFileName) behind the scenes.

function process(onData:Bytes ->Int ->Int ->Surprise<Noise, Error>?partSize:Int = null):Surprise<Noise, Error>

A method for streaming data to a specified method.

Will read the temporary file from the disk, one part at a time. Each part that is read will be passed to the "onData" function. Once onData's future is resolved, the next part will be written.

onData
  • method to execute for each set, eg. function onData(data:Bytes, pos:Int, length:Int):Surprise<Noise,Error>
partSize
  • the maximum amount of data to stream in each part. Default is 8KB for PHP, 16KB for other targets.
returns

a future to notify you once all the data has been processed, or if an error occured at any point.

function writeToFile(newFilePath:String):Surprise<Noise, Error>

Write the current upload to a file on the filesystem.

Uses sys.io.File.copy( tmpFileName, newFilePath ) behind the scenes.
