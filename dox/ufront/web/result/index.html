package ufront.web.result

ActionOutcome

A typedef shortcut for an action return type that either gives a valid result or a Error

ActionResult

ActionResult is the base class for all results of actions performed during a MVC request.

ContentResult

Represents a user-defined content type that is the result of an action method.

EmptyResult

Represents a result that takes no further action and writes nothing to the response.

FutureActionOutcome

A typedef shortcut for a Future that will contain either an ActionResult or a Error

FutureActionResult

A typedef shortcut for a Future that will contain an ActionResult

RedirectResult

Controls the processing of application actions by redirecting to a specified URI.

ViewResult

A ViewResult loads a view from a templating engine, optionally wraps it in a layout, and writes the result to the HttpResponse with a text/html content type.
