class ufront.web.result.EmptyResult extends ActionResult

Available on all platforms

Represents a result that takes no further action and writes nothing to the response.

This is useful if your controller action has written to the response directly, for example using Sys.println.

@author Andreas Soderlund

preventFlush

Should we prevent the application from "flushing" the response, this will prevent both content and headers, cookies, status codes etc from being returned to the client.
