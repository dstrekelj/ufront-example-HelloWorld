interface ufront.app.UFMiddleware extends UFRequestMiddleware

Available on all platforms

Implementors
InlineSessionMiddleware, TmpFileUploadMiddleware
InlineSessionMiddleware

Make sure we init() sessions before the request starts, and commit() them before it ends

TmpFileUploadMiddleware

If the HttpRequest is multipart, parse it, and store any uploads in a temporary file, adding them to httpRequest.files

An interface representing Middleware that is applied both before and after processing the request.

This can be useful if you have middleware that sits in both positions

See UFRequestMiddleware and UFResponseMiddleware for more details.
