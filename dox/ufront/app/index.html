package ufront.app

HttpApplication

The base class for a HTTP Application

HttpApplicationMacros

UFErrorHandler

Represents an error handler that can help us output, log, diagnose or even recover from errors in our HttpApplication.

UFInitRequired

This is to be used with Handlers or Middleware that require an init() function.

UFLogHandler

Interface for a Logger - something which takes trace, log and error messages from the request and logs them appropriately.

UFMiddleware

An interface representing Middleware that is applied both before and after processing the request.

UFRequestHandler

Represents an object that can handle a HttpRequest, process as required, and write the HttpResponse.

UFRequestMiddleware

Middleware that runs before the request has been processed.

UFResponseMiddleware

Middleware that runs after the request has been processed.
