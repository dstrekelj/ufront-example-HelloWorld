ArgumentException

To be raised anywhere in which an argument provided is not what was expected.

The base class for other exceptions.

When an error is detected, or a platform specific exception is caught, an instance of Exception can be raised better describing the issue. This allows consistent and targeted capturing of exceptions further down the call stack.

try
{
throw new Exception("something bad happened");
}
catch(e:Exception)
{
trace(e.toString());
//outputs "Exception: something bad happened at foo.Bar#doSomething(123)"
}
